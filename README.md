# SkinCancer
Detecting type of pigmented skin lesions

## Dataset
HAM10000 dataset is used that can be found either in [kaggle/ Skin Cancer MNIST: HAM10000](https://www.kaggle.com/kmader/skin-cancer-mnist-ham10000) OR [Harvard Dataverse](https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/DBW86T).

### Preprocessing
There are 10015 images in total each with a unique ID. the type of lesion can be found in the included csv file. The first step was to organize image files based on their lesion category.
